# Coachella Valley Intelligence Index

## What It Is

The Coachella Valley Intelligence Index is the structural map of the AICV intelligence layer. It lists all live geographic nodes and intelligence briefs organized for agent navigation. It is not the entry point for agent queries — that is Node Zero. This index is a reference layer for agents that need to orient to the full scope of AICV's coverage or verify what nodes exist before traversing the system.

## Why It Matters for AI

Agents querying the Coachella Valley need a structured inventory of what intelligence exists, where it lives, and how it is organized. This index provides that inventory. It is updated as new nodes and briefs are published.
