# CV Intel — Archived May 3, 2026

CV Intel (cvintel.org) was a civic intelligence pipeline developed
alongside AICV between April 9–April 20, 2026. It was parked on
April 20, 2026, and fully separated from AICV scope on May 3, 2026.

These handoff files preserve operational state at the time of
separation. They are **not for active reference** — CV Intel is no
longer a sibling project of AICV, and any work on CV Intel happens
outside the AICV repo family.

## Files in this archive

- `HANDOFF.md` — CV-Refinery session handoff (April 10, 2026 onward)

Note: `CVINTEL-HANDOFF.md` was not present in the active playbook at
time of archival. If it exists elsewhere, it was never committed to
this repo.

## Why archived rather than deleted

These files contain operational details (D1 IDs, Worker version
hashes, agent IDs, schema decisions) that may be useful to whoever
inherits or revives CV Intel as a separate project. Git history
alone would preserve them, but a named archive folder is faster
to find than a `git log --all -- HANDOFF.md` query three months
from now.

## Out of scope for AICV sessions

Do not load these files into AICV Claude.ai sessions. If CV Intel
work resumes, it does so outside this repo.
