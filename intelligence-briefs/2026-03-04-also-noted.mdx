## Signals

### GPT-5.4 Leaked One Day After GPT-5.3 Launch

One day after releasing GPT-5.3, OpenAI posted five words on X — "5.4 sooner than you think" — triggering widespread speculation. Leaked signals from code commits, screenshots, and API listings point to three reported capabilities: an extreme reasoning mode that allocates more compute to harder problems, a 2 million token context window (up from roughly 128k in current models), and pixel-level image analysis improvements. Prediction markets give a 55% probability of release by April 2026. Nothing is officially confirmed. The signal for the AI economy is OpenAI's sprint cadence — multiple model iterations per month, with the gap between releases shrinking fast.

### OpenAI Building Its Own GitHub Competitor

OpenAI is reportedly developing a native code repository platform to replace its dependence on GitHub, owned by Microsoft. The move follows a series of GitHub outages in late February and early March 2026 that disrupted OpenAI's model training pipelines and AI agent workflows. The underlying tension is significant: Microsoft has invested $13 billion in OpenAI while simultaneously building competing frontier AI models, blocking OpenAI's attempted $3 billion acquisition of coding startup Windsurf, and facing antitrust complaints from OpenAI. A native OpenAI code platform would integrate directly with Codex — OpenAI's coding environment with over 1 million developers — enabling AI agents to summarize code, detect security issues, and manage workflows without leaving the platform. GitHub currently hosts over 420 million projects; developer migration would be gradual but the directional signal is clear: OpenAI is moving to control its full stack.
